# プログラムを書くにあたって、どういう順番でやったのかを示します。
## 仮想環境を作成する

``` bash
python -m venv ll_env
```

## 仮想環境を有効化する

``` bash
source ll_env/bin/activate
```

Windowsの場合

``` bash
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser 
ll_env\Scripts\activate
```

## Djangoをインストールする

```bash
pip install --upgrade pip
```
